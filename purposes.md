阶段 0：准备环境
├─ 安装 MinGW / GCC
├─ 安装 CLion 并配置 Toolchain
├─ 下载 PDCurses 并编译静态库
└─ 配置 CMake 链接 PDCurses

阶段 1：最小可运行版本 (MVP)
├─ 游戏逻辑：
│   └─ 定义 Tetromino 数据结构（形状矩阵、位置）
├─ 控制台显示：
│   └─ 使用 PDCurses 打印一个方块
├─ 输入处理：
│   └─ PDCurses 捕获键盘左右移动
└─ 主循环：
    └─ 下落方块 + 屏幕刷新（refresh）

阶段 2：基本玩法完善
├─ 方块旋转

├─ 计分系统
